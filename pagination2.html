WHAT'S PAGINATION?

Pagination is a process of dividing large amount of content into discrete pages. Instead of showing 100 images at once you can show 10 images per page.

STRUCTURE:

The pagination is built using an unordered list , which is placed inside the "nav". The "ul" class is pagination, each list item has the class "page-item" and inside each list item we put the "a" tag in which we use the class "page-link". The "li" can have the active class to show which page the user is currently on and it turns blue ; it can also be disabled , it's not clickable.
