CSS:

Css works as an interiror designer, with the help of css we can style our web.

We have the "selector rule", it has a selector(is the HTML element that we want to style); property(is the feature we want to change);value(is the specific setting we want to apply).

WAYS TO ADD CSS:

INLINE: we put the style directly inside the HTML tag.

INTERNAL: we put the style tag on top of the html file.

EXTERNAL: we create a separate css file and link it to our HTML.

BOX MODEL:

every single element on our webpage is a rectangular box. To control it's size and spacing we use padding, margin, border...
